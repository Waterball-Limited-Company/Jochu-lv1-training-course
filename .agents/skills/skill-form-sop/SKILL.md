# SOP

## Phase 1 -- 收斂並完成 SOP

1. READ 讀取目標 skill 內容，確認要新增、修改或重寫的 SOP 範圍。
2. READ 讀取 `rules/sop-格式規範.md`，確認 SOP 的基本格式要求。
3. THINK 先依已載入規則整理 phase 與 step；若需要判斷步驟是否過長、是否混入前情提要，或是否應延後載入規則，再讀取 `rules/SOP-精簡與按需載入.md`，並依本次已載入規則收斂最小可用的 SOP。
4. WRITE 依本次已載入規則撰寫或修改目標 skill 的 SOP section。
5. READ 回頭檢查結果是否符合本次已載入規則；若不符合，立即修正。
